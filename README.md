# 📊 Bank Transaction & Fraud Analysis Dashboard

## 📌 Project Overview
Interactive Power BI dashboard built on 1M+ real-world bank transactions 
to visualize revenue trends, fraud patterns, customer segmentation 
and top performing cities.

## 🛠️ Tools Used
- Power BI Desktop
- MySQL (data source)
- Python / Jupyter Notebook (data cleaning)

## 📊 Dataset
- 1,048,575 transactions
- 2,000 customers
- 8,914,963 fraud labels
- Source: Kaggle Financial Transactions Dataset

## 📈 Dashboard Visuals
- 💳 KPI Cards — Total Transactions, Total Revenue, Avg Transaction Value
- 📉 Line Chart — Revenue Trend by Year
- 🍩 Donut Chart — Customer Segmentation (High/Mid/Low Value)
- 📊 Bar Chart — Top 10 Cities by Revenue
- 📊 Bar Chart — Transactions by Payment Type
- 🔍 Tooltip — Hover insights on line chart

## 💡 Key Insights
- 💰 Business generated $45.9M across 1M+ transactions
- 🌐 Online transactions have 88x higher fraud rate than swipe
- 👴 46-60 age group is the highest value customer segment
- 🏙️ ONLINE channel dominates revenue at $7M+
- 📉 Revenue shows gradual decline since 2002
